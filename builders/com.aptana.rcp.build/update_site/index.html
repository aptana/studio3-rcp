Aptana Studio Update Site

This site is designed to be used through the Aptana Studio update manager.

Update Aptana Studio

  1. From the Help menu, select Install New Software... to open an Install pop-up window.
  2. In the Work with: text box of the Install window, type the URL for the update site, and hit the Enter key.
  3. Close the Install window.
  4. From the Help menu, select Check for Updates.
  5. In the new window, click the Next button and then click the Next button again.
  6. Choose the option to accept the terms of the license agreement, and click the Finish button.

Manual Installation

Download Plugin Update Site
  1. Save the above file to an easy-to-find location.
  2. Open Aptana Studio, and go to Help -> Install New Software....
  3. Click the Add... button to open the Add Site window.
  4. Click the Archive... button, and select the file saved in step 1.
  5. Close the Install window.
  6. From the Help menu, select Check for Updates.
  7. In the new window, click Next -> Next.
  8. Choose the option to accept the terms of the license agreement, and click the Finish button.
